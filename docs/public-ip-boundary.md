# Public IP Boundary

This repository is intentionally generic.

It may demonstrate:

- event-driven design
- local-first logging
- metadata-aware export
- feature extraction scaffolding
- provenance-preserving correction/redaction concepts

It must not include:

- private logs
- proprietary schemas
- production datasets
- internal project names
- private architecture diagrams
- real user/device identifiers

Use synthetic sample data for public demos.
